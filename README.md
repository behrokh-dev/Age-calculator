# 🎂 Age Calculator

A simple and beginner-friendly **Age Calculator** built with **HTML, CSS, and JavaScript**.

This project was created as one of my first JavaScript exercises to practice working with dates, DOM manipulation, and user input handling. It calculates the user's age based on the entered birth year.
